# AI圆桌 · 成长对话系统

一个基于AI的个人成长对话平台，支持多AI角色协同对话，帮助用户进行自我反思和成长。

## 功能特色

### 🎭 四位AI伙伴
- **💡 Coach 小柯**: 成长引导者，温和提问式引导思考
- **🧠 Strategist 老谋**: 战略顾问，理性分析和框架思维
- **💖 Psychologist 心理姐**: 情绪伙伴，共情支持和心理疏导
- **⚡ Operator 阿操**: 落地专家，提供具体行动方案

### 🌟 核心功能
- **AI圆桌对话**: 多AI角色协同，提供多维度建议
- **智能角色切换**: AI根据对话内容智能选择最适合的角色回复
- **思考链可视化**: 点击查看AI的思考过程
- **成长日志系统**: 记录对话历程，可视化成长数据
- **成长树养成**: 通过对话互动培养虚拟成长树
- **全局搜索**: 快速查找历史对话内容

### 🎨 设计理念
- **温暖专业感**: 脱离冷办公气质，使用温暖色调
- **企业级交互**: 借鉴企业微信/钉钉的高效导航结构
- **功能性动画**: 引导型动画，提升用户体验

## 技术架构

### 前端技术栈
- **HTML5 + CSS3**: 基础页面结构和样式
- **Tailwind CSS**: 快速样式开发
- **Vanilla JavaScript**: 轻量级交互逻辑
- **本地存储**: 使用localStorage保存用户数据

### 支持的AI平台
- **阿里云百炼**: 通义千问系列模型
- **OpenRouter**: 支持多种主流AI模型

## 快速开始

### 1. 启动应用
```bash
# 方法一：使用Python启动（推荐）
python start_server.py

# 方法二：直接打开HTML文件
# 在浏览器中打开 index.html
```

### 2. 配置AI服务
1. 点击左下角"⚙️ 配置中心"
2. 选择AI平台（阿里云百炼 或 OpenRouter）
3. 输入对应的API Key
4. 选择文本模型
5. 保存配置

### 3. 开始对话
1. 点击"+ 新建AI圆桌"
2. 选择想要参与的AI角色
3. 创建圆桌后开始对话
4. AI会根据内容智能选择角色回复

## API Key 获取

### 阿里云百炼
1. 访问 [阿里云百炼控制台](https://dashscope.console.aliyun.com/)
2. 创建应用并获取API Key
3. 支持模型：qwen-plus, qwen-max, qwen-turbo

### OpenRouter
1. 访问 [OpenRouter官网](https://openrouter.ai/)
2. 注册账号并获取API Key
3. 支持多种模型：GPT-4, Claude, Qwen等

## 使用说明

### 创建圆桌对话
- 可以选择1-4个AI角色参与对话
- 每个角色都有独特的专业领域和回复风格
- 系统会根据对话内容智能选择最合适的角色回复

### 查看思考过程
- 每条AI回复都带有💭图标
- 点击可展开查看AI的内部思考链
- 了解AI是如何分析问题和生成回复的

### 成长日志功能
- 自动记录所有对话数据
- 可视化展示成长统计
- 成长树会随着互动次数升级
- 支持按时间和标签筛选历史记录

### 搜索功能
- 支持全局搜索历史对话
- 可以按关键词快速定位相关内容
- 搜索结果会高亮显示匹配内容

## 数据安全

- **本地存储**: 所有对话数据存储在浏览器本地
- **API Key安全**: API Key仅在本地存储，不会上传到任何服务器
- **隐私保护**: 对话内容仅在用户设备和选择的AI服务商之间传输

## 自定义配置

### 动画设置
- 可在配置中心关闭所有动画效果
- 适合对动画敏感的用户

### 角色自定义
- 支持重命名AI角色（如改为"老铁"、"大树"等）
- 可设置默认角色组合
- 未来版本将支持自定义角色头像

## 故障排除

### 常见问题

**Q: AI不回复或报错**
A: 请检查：
1. API Key是否正确配置
2. 网络连接是否正常
3. 选择的模型是否支持
4. API Key是否有足够余额

**Q: 对话记录丢失**
A: 对话记录存储在浏览器本地，清除浏览器数据会导致丢失。建议定期导出重要对话。

**Q: 页面显示异常**
A: 请尝试：
1. 刷新页面
2. 清除浏览器缓存
3. 使用Chrome或Edge等现代浏览器
